🔧 Setup Guide

Quick Start

  1. Run supabase_schema.sql in Supabase SQL Editor
  2. Go to ⚙️ Settings → fill LiveKit credentials → Save
  3. Fill Google Gemini API Key → Save
  4. Fill Vobiz SIP details → Save → click ⚡ Create SIP Trunk
  5. Go to 🤖 Agents → create an agent profile
  6. Go to 📞 Single Call → test with your number
  7. Deploy via Coolify using the Dockerfile
